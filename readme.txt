\h 絹代
ミツツボアリを養殖している養蟻家で、直売店の運営も
している。アリーのことを女王蟻だと思い、ご利益が
あると思って留まってもらっている（軟禁ともいう）。

\u アリー
行き倒れていた所を絹代に保護された夢魔。サキュバス。
絹代には女王蟻だと思われている。触覚が生えてるから。
